Key Design Decisions

Tabs

Sales / Rentals toggle with count badges. Each tab has its own pipeline stages.

Pipeline Summary Bar

Clickable status pills that double as filters. Click "Offer Received" to show only those.

Card Layout

3-column grid on desktop, 2 on tablet, 1 on mobile. PF-style: photo → price → location → specs → status.

Filter Panel

Area, bedrooms, price range, property type, status. Matches how PF/Bayut filter.
